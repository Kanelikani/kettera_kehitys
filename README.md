Koirien Tinder

Koiranomistajille suunnattu mobiili-/web-sovellus, jossa voit luoda omalle koirallesi profiilin, nähdä toisten käyttäjien luomat profiilit ja ”matchata” niiden kanssa luoden uusia kontakteja. Uuden kontaktin luotuasi voit lähettää toisille käyttäjille viestejä, ja käydä koirien leikkitreffeillä. 

Käyttäjä
-	Tilin luonti
-	Kirjautuminen sisään
-	Kirjautuminen ulos
-	Koiran profiilin luonti (koiran profiilissa saatavilla kirjoitettu kuvaus koirasta, kuva, rotu, tms)
-	Koiran profiilin poisto 

Matchaus
-	Toisten käyttäjien luomien koirien profiilien selaaminen
-	Funktionaalisuus tykätä toisen profiilista
-	Jos molemmat käyttäjät pitävät toistensa profiileista, avautuu mahdollisuus viestinnälle

Viestintä
-	Käyttäjien keskeinen viestintä matchauksen jälkeen
